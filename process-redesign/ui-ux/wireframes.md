# HPCL Procurement System - UI/UX Wireframes

## Overview
This document provides text-based wireframes (ASCII art) for all major screens in the HPCL Procurement Automation System, designed for desktop and mobile responsiveness.

---

## Design Principles
1. **HPCL Brand Consistency**: Official color palette (#003366, #E4002B, #FFB400)
2. **Accessibility**: WCAG 2.1 AA compliance, keyboard navigation, screen reader support
3. **Mobile-First**: Responsive design for tablets and smartphones
4. **Role-Based Views**: Customized dashboards for Buyers, Approvers, Managers, Finance
5. **Real-Time Updates**: Live status indicators, notifications, progress bars
